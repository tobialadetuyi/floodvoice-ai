# FloodVoice

FloodVoice is an open-source early-warning platform that combines satellite data and community observations with AI to protect climate-vulnerable communities in low‑resource settings.

## Overview
FloodVoice forecasts short-term flash floods (6–12 hours) using an LSTM model that fuses:
- NASA GPM rainfall
- Sentinel-1 flood extent
- Community river-gauge observations delivered via USSD

It produces local-language voice alerts (autogenerated TTS) for low-bandwidth delivery (IVR, WhatsApp audio) and can run offline on low-cost hardware (e.g., Raspberry Pi).

## Problem
- Increasing flash floods displace thousands yearly in low-lying Least Developed Countries (LDCs) and Small Island Developing States (SIDS).
- National alerts often arrive in English, require mobile-data, and reach communities too late.
- The 2024 floods in Magboro–Akeran highlighted the need for local-language, basic-phone warnings.

## Solution
- Short-range AI forecast (6–12h) using an LSTM that fuses remote sensing and community data.
- Local voice alerts in languages such as Wolof, Kriolu, Mandinka, and Yoruba.
- 2G delivery via IVR and WhatsApp audio; system supports offline operation on a Raspberry Pi.

## Features
- Multimodal input: satellite + in-situ sensors + community reports
- Short-term LSTM forecasting
- Auto-generated TTS in multiple local languages
- IVR/USSD integration for basic phones
- Dockerized deployment for edge devices (Raspberry Pi support)
